# LightAsync
Lightasync aims to be a fast, lightweight, cross platform and multi gui toolkit
library for asynchronous IO. It is written in Python and using Enhanced
Generators as described in PEP 342.

## Working
 * Platforms:
 	* GLib
 	* Epoll
